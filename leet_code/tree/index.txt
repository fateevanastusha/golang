
Листы - ноды, у которых нет детей.
Родитель  
Ребенок 
Бинарное дерево - у каждой ноды максимум 2 ребенка.
Нода, узел - элемент дерева.
Корень - верхняя нода дерева (у которого нет родителей).
Корневое дерево - дерево, у которого мы знаем и имеем корень.
Поддерево - часть дерева, которая сама по себе является деревом.
